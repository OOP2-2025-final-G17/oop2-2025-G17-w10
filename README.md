# アプリ名: oop2_10_sample
> アプリ名は、プロジェクトの顔でもあるので、適切な命名を行ってください。

概要:
> ユーザー・職場・労働時間を登録し、シフト管理と集計を行うことができるwebアプリケーションです。
シフト登録をすると、当日のシフト、月ごとの就労時間、給与をダッシュボード形式で表示できます。

## アピールポイント

### ・シフト管理に特化したシンプルな設計
ユーザー・職場・労働時間を分けて管理することで、複数人のシフトを管理しやすくなっています。

### ・登録したシフトを１画面で把握できるダッシュボード
当日と月間のシフト、月毎の就労時間と給与、職場ごとの人数を表示し、使用者が現在のシフト状況や人員配置を直感的に把握できるようにしています。
